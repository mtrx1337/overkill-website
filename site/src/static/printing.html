Printing (you probably know the struggle)

First of all, you have to install the CUPS package. This is the huge printing framework that you WILL need. It provides 99% of the printing drivers and software. In my case the framework is not enough, because I have a quite old and exotic Epson Inkjet printer. If you too have an old Epson printer, install the package "epson-inkjet-printer-escpr" from the AUR. I suggest first trying the steps below. If you can't get your printer to work, maybe look up your printers driver in the AUR!

Installing the CUPS framework:
sudo pacman -S cups cups-pdf
For Epson owners:
yay -S epson-inkjet-printer-escpr

You can replace "yay" with any AUR manager you want, I just happen to use yay. The next package you will probably want to have is "system-config-printer". This is the GNOME printing manager as far as I know. It allows you to easily enable your printer for system wide printing. This package is optional if you know what you're doing.

Optional:
sudo pacman -S system-config-printing

To be able to use CUPS, you have to enable and start it's service now. This can be done with a one line command.
sudo systemctl enable org.cups.cupsd.service && sudo systemctl start org.cups.cupsd.service

Finally, you have to visit this link:
http://localhost:631/
It may look intimidating at first, at least it was for me. Like, what now? Anyways. This is your CUPS webinterface from where you can add your printer. Looking at the middle, you can see the headline "CUPS for administrators". Below you should see the link "Adding Printers and Classes". Click this. Afterwards you should see "Add Printer" in the top left corner. There you have to log in with your linux user name and password. Now select the printer you want to add and set up. Hit continue and you're done!
If you want to can check out the optional package that I spoke about earlier. It allows for easy managing of the printing and group settings.
